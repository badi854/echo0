## 📖 简介

**Echo 0** 是一个纯前端、零依赖的单页 Web 应用。它拒绝被定义为社交产品——没有点赞、没有算法推荐、没有已读回执。每一条消息都是投向自己时间线的一颗石子：落下，留痕，然后成为历史的一部分。

**它不是一个工具，而是一座回音壁。**

---

## ✨ 特性

### 🔮 核心体验
- **CRT 启动/关机动画** —— 老式阴极射线管开机仪式感
- **终端打字机音效** —— 实时合成，无需任何音频文件
- **实心方块光标** —— 拒绝现代细竖线，回归终端本色
- **深空炭黑主题** + **琥珀/翠绿** 两种复古配色

### 📝 消息管理
- 发送 / 编辑 / 置顶 / 撤回（坍缩）/ 焚毁
- 撤回 ≠ 删除 —— 保留时间伤疤：`<< 此信息已坍缩 >>`
- 每条消息拥有**量子指纹** —— 基于文本长度生成的独特几何图形

### 🏷️ 标签与筛选
- 消息内使用 `#标签` 自动添加
- 点击标签自动筛选该主题
- `/filter <标签>` 精准过滤时间线

### 📊 统计与回顾
- `/analytics` —— 总消息数、总字数、日均消息、高频词汇 Top 5、今日消息数
- `/calendar` —— 最近 30 天 ASCII 热力图，点击日期跳转
- `/random` —— 随机回溯一条消息，声呐脉冲高亮

### 💾 数据主权
- 所有数据存储在 `localStorage`，永不离开你的设备
- `/export` —— 完整导出宇宙（JSON 格式）
- `/import` —— 一键恢复备份，跨设备迁移

### ⚡ 终端命令系统
- 支持 **命令别名**（如 `/s` = `/search`，`/e` = `/edit`，`/p` = `/pin`）
- 输入 `/` 自动联想匹配命令
- 命令列表：搜索、编辑、置顶、跳转、标签、主题、导出/导入、音效开关等

### 🎨 文本格式
- `**粗体**` —— 醒目强调
- `*斜体*` —— 优雅修饰
- `~~删除线~~` —— 时间痕迹

### 🔇 音效开关
- `/sound` 一键切换按键音与发送音，偏好自动保存

### 📱 移动端适配
- 触屏长按消息呼出菜单
- 响应式布局，手机/平板/桌面全支持

## 🎯 设计哲学

1. **零外部依赖**  
   不依赖任何第三方库、框架、CDN 资源。一个 HTML 文件，打开即用。

2. **零图片资源**  
   所有图形（背景粒子、量子指纹、身份印记、网络图标）均由 Canvas 在运行时现场绘制。

3. **零后端**  
   所有用户数据（时间线、身份、共振关系）只存在于 localStorage。没有服务器，没有数据库，没有隐私泄露风险。

4. **无状态协议**  
   页面刷新即重置所有运行时变量，仅从 localStorage 静默拉取历史，无任何自动保存提示。

5. **时间线哲学**
   - 撤回 ≠ 删除 —— 坍缩后的消息留下可见的时间伤疤
   - 编辑 ≠ 覆盖 —— 编辑后的消息显示 `[edited at ...]`
   - 焚毁 ≠ 忘记 —— 清屏归档时，消息以坠落动画沉入底部
